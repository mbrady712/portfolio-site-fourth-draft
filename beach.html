The Amy Beach Piano Foundation

This site was my final project for my Intermediate WWW Authoring, HTML, and CSS course. In this project, students had to design a website for a fictional organization. I came up with the idea of a foundation for the piano works of the American composer Amy Beach. The site features Beach's biographical information, a table of her piano works, and a donation page with a form for collecting credit card information. The form data doesn't submit anywhere, as HTMl and CSS were the only languages that we could use on this project. The site can be viewed on ACC's student web server.
